2. SISTEMA DE ALERTAS (3 TIPOS PRINCIPALES)

   ┌─ 🔴 ALERTA POR CALOR EXTREMO
   │  ├─ Nivel Crítico: T > 45°C
   │  ├─ Nivel Alto: T > 40°C
   │  └─ Recomendaciones: Protocolo de emergencia
   │
   ├─ 🔴 ALERTA POR VIENTO PELIGROSO
   │  ├─ Nivel Crítico: V > 70 km/h
   │  ├─ Nivel Alto: V > 50 km/h
   │  └─ Recomendaciones: Cerrar zonas
   │
   └─ 🟡 ALERTA POR LLUVIA/HUMEDAD
      ├─ Lluvia activa detectada
      ├─ Humedad muy alta (> 95%)
      ├─ Humedad muy baja (< 20%)
      └─ Recomendaciones: Monitoreo de drenaje

3. VALIDACIÓN ROBUSTA DE INPUTS

   • Temperatura: -50 a 60 °C
   • Humedad: 0 a 100 %
   • Viento: 0 a 200 km/h
   • Fecha: DD/MM/AAAA
   • Distrito: 21 distritos válidos
   • Lluvia: S/N con reintentos ilimitados
   • Sin ingreso vacío permitido
   • Mensajes claros ante errores

🚀 CÓMO USAR

INICIO RÁPIDO:
1. Activar entorno:    .\.venv\Scripts\Activate.ps1  (PowerShell)
2. Ejecutar programa:  python main.py
3. Seleccionar opción: Ingresa 1, 2, 3, 4 o 5

REGISTRAR UN DATO:
1. Seleccionar opción 1
2. Ingresar fecha (ej: 25/03/2026)
3. Seleccionar distrito (ej: Centro)
4. Ingresar temperatura (ej: 42)
5. Ingresar humedad (ej: 65)
6. Ingresar viento (ej: 55)
7. ¿Lluvia? S/N
8. Revisar resumen
9. El SISTEMA ANALIZA ALERTAS automáticamente
10. Confirmar guardado (S/N)
11. Datos guardados en datos_clima.json
